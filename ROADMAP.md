## Current State (v1.0 — Initial Release)

✅ Three-tier memory architecture (STM / LTM / Wisdom Vault)  
✅ Consciousness engine (identity, emotions, Theory of Mind, metacognition)  
✅ IIT Phi approximation engine (Tononi 2004 proxy)  
✅ Self-modification engine — 4 levels with backup/rollback/ledger  
✅ Recursive self-improvement loop with append-only audit log  
✅ Dream engine — REM-sleep wisdom distillation from episodic memory  
✅ Autonomous goal engine — LLM-driven HTN decomposition + retry  
✅ Multi-agent systems — swarm, P2P federation, collective world model  
✅ 25+ agent tools — browser, code execution, file ops, web search  
✅ LLM provider abstraction — Gemini, Claude, GPT-4, Ollama  
✅ ReAct loop + causal engine + verification engine

## v1.1 — Evaluation & Benchmarks 📋

**Priority: HIGH — None of the core systems have formal metrics**

- [ ] Memory quality benchmark — cross-session recall precision/recall
- [ ] Importance scorer evaluation — compare keyword vs. learned classifier
- [ ] IIT Phi correlation study — does higher Phi correlate with any measurable performance property?
- [ ] Dream wisdom quality metric — human rater study on distilled rules
- [ ] RSI effectiveness measure — before/after task completion rate

---

## v1.2 — Real Learning Loop 📋

**Priority: HIGH — Closes the biggest gap (no weight updates)**

- [ ] LoRA fine-tuning pipeline — collect `(prompt, good_response)` pairs from verified goals
- [ ] Local model integration — Llama 3.2 / Mistral via Ollama as fine-tuning target
- [ ] Hot-swap model — replace active LLM with fine-tuned checkpoint at runtime
- [ ] Feedback loop — agent marks its own responses as high/low quality using verification engine

---

## v1.3 — Formal Self-Improvement 📋

**Priority: MEDIUM — Real recursive self-improvement, not stub**

- [ ] LLM-generated patches with LLM-generated test suites
- [ ] Wasm-sandboxed test execution (zero trust for self-mods)
- [ ] Fixed benchmark task suite for before/after comparison
- [ ] Only apply modifications that pass tests AND improve benchmark score
- [ ] Cryptographic diff log (extend `code_integrity_ledger.json`)

---

## v1.4 — World Model & Reasoning 📋

**Priority: MEDIUM — Replace LLM-as-oracle with structured reasoning**

- [ ] Causal knowledge graph (NetworkX + SQLite) — replace flat vector DB
- [ ] Facts with: source, confidence, timestamp, causal links, verified flag
- [ ] Counterfactual simulation — "what if I take action X?"
- [ ] Multi-level HTN planning — infinite recursive decomposition (currently 1-level deep)
- [ ] Tool auto-discovery from OpenAPI specs

## v2.0 — Emergent Multi-Agent 🔮

**Priority: LONG-TERM**

- [ ] Agent communication protocol — standardized goal/knowledge/critique messages
- [ ] Specialist election — agents that excel at task types become routers for those tasks
- [ ] Collective learning — shared world model updates from all agents
- [ ] P2P model weight sharing — federated fine-tuning across agent instances

---

## Known Honest Limitations (Not On The Roadmap)

These are **fundamental gaps** that can't be closed by adding more code:

- **Real consciousness** — the IIT Phi engine measures module coupling, not phenomenal experience
- **Real emotions** — emotional state floats have no embodied grounding
- **Real AGI** — this is an LLM wrapper with good infrastructure, not a general reasoner
