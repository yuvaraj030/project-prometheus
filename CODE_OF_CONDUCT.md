# Code of Conduct

## Our Pledge

This is an open research project. Everyone who participates — whether filing issues, writing code, or discussing ideas — is expected to engage in good faith and treat others with respect.

## Standards

**Encouraged:**
- Honest, rigorous technical discussion
- Constructive criticism of ideas (including the project's own claims)
- Sharing related research, papers, or experiments
- Asking questions at any skill level

**Not acceptable:**
- Personal attacks or harassment
- Dismissing contributions without explanation
- Overstating capabilities or spreading misinformation about AI
- Spam or self-promotion unrelated to the project

## Scope

This code of conduct applies in all project spaces — GitHub issues, pull requests, discussions, and any other official channels.

## Enforcement

Violations can be reported by opening an issue marked `[conduct]` or contacting the maintainer directly via GitHub. Repeated violations will result in being blocked from the repository.

---

*This project believes the AGI research community is best served by honesty, rigor, and mutual respect.*
